# Nature is Beautiful \[rebalance\]

A copy of [this](https://steamcommunity.com/sharedfiles/filedetails/?id=1530259312) with some altered values which fits more how i think it should be balanced.

## Getting Started

* Subscribe to the mod [here](https://steamcommunity.com/sharedfiles/filedetails/?id=1551147619)
* Open the mod list and activate it
* Make sure it is *after* the dependency [HugsLib](https://steamcommunity.com/workshop/filedetails/?id=818773962)

### Prerequisites

* [HugsLib](https://steamcommunity.com/workshop/filedetails/?id=818773962)
